# hc-analytics-plugin

Google Analytics, custom events, and metrics for the Apito API platform.

## Features

- Google Analytics 4
- Custom event tracking
- User behavior analysis
- Conversion tracking
- Real-time reports

## Installation

```bash
go build -o hc-analytics-plugin .
```
